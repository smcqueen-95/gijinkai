# Site Structure for Gijinkai Studio

## Pages
1. Home
   - Hero section with company name and tagline
   - Featured projects showcase
   - Services overview
   - Call to action

2. About
   - Company overview
   - Our Approach section
   - Team members
   - Careers section

3. Projects
   - Game Development Projects
   - Web/App Development Projects
   - Manga and Anime Creation

4. Services
   - Service categories
   - Process overview
   - Technologies used

5. Products
   - Software products
   - Digital assets
   - Featured products

6. Pricing
   - Service packages
   - Custom solutions
   - Comparison table

7. Contact
   - Contact form for new clients
   - Office locations
   - Social media links

## Navigation Structure
- Main navigation: Home, Projects, Services, Products, About, Pricing, Contact
- Footer navigation: All main pages plus privacy policy, terms of service
- Mobile: Hamburger menu with all main navigation items

## Component Structure
- Header with logo and navigation
- Footer with contact info, sitemap, and social links
- Page layout wrapper
- Hero section component
- Card components for projects and services
- Contact form component
- Gradient button component
- Japanese-inspired decorative elements
